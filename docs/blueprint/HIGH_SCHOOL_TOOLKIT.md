# UEM 고교/전통 수학 툴킷 패키지 (정의 고정안)

> 목표: 미적분/부등식/연속성/삼각/급수 등 전통·고교 문제에 UEM+한글 연산자를 적용할 때 필요한 표준 연산자 세트, 여백 필터, 동치 정리를 고정하여 “증명만 하면 되는” 상태로 끌어올린다.

## 1. 표준 연산자 세트 (C,V,F 조합 예시)
- 미분/적분: ∂, ∫, 평균값, 극값, 단조성 검사.
- 정의역 필터: 루트/로그/분수/삼각 함수 정의역을 여백으로 보내는 필터 연산자.
- 사영: 삼각함수를 단위원/복소평면으로 사영하는 변환, 좌표축/직선/원 사영.
- 여백 로그: 필터/사영 시 MarginLog 갱신을 의무화.

## 2. 여백 제거 자동 필터
- must/optional/forbidden 주석 키를 정의하고, 병렬 전개(⊗_par) 후 주석을 읽어 실행 체인을 결정하는 규칙을 수식으로 명시.
- 유형별 필터 예:  
  - √(p(x)) 문제: p(x)≥0 조건을 must 필터로, 나머지 스텝 optional.  
  - 로그 ln(p(x)): p(x)>0 must 필터.  
  - 분수 f/g: g≠0 must 필터.

## 3. 예제/동치 정리 (문장 형식)
- 최소 대상: 연속성/미분가능/극값/최소·최대, 삼각 합성, 완전제곱/부등식, 수열 극한/급수.  
- 각 문제 유형별 동치 정리: “기존 풀이 ↔ UEM+한글연산자 풀이” (조건: 다항/유리/삼각 다항식 등)와 정의역·필터·사영 조건을 명시.
- “풀이 스텝 템플릿”을 문제 유형별로 1개 이상 고정(정의역 필터 → 사영/변수치환 → 미분/적분 → 극값/부등식 판단).

## 4. HS 서브셋 규약
- HS에서 허용되는 자모/연산자/규칙 서브셋을 표로 명시(∇_hangul 2~3 성분, 제한된 자모).  
- HS 서브셋에서 종료성/합류/타입 보존이 성립함을 별도 lemma로 목표화.

## 5. 검증 체크리스트
1) 표준 연산자 세트(C,V,F)와 여백 로그 갱신 의무 명시.  
2) must/optional/forbidden 필터 규칙 수식화.  
3) 문제 유형별 동치 정리 문장(조건 포함) 고정.  
4) 스텝 템플릿 최소 1개씩 작성.  
5) HS 서브셋 표와 종료성/합류/타입 보존 lemma 목표 명시.
