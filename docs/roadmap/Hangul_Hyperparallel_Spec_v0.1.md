# Hangul Hyperparallel Operators for UEM (Spec v0.1)
> 목적: UEM에서 한글 음절을 초성/중성/종성 연산자 튜플로 해석해 초병렬 연산(⊗_par, ∇_hangul)을 지원하기 위한 최소 스펙을 고정한다.  
> 범위: 문서 스펙. Lean 증명 및 코드 매핑은 후속 작업.  
> 원천 참고: LEGACY UEM_YJG 문서(초/중/종성 역할), JIWOL LUT(초/중/종성 표), gg_radix 엔진(분해/조합 코드) 등. **이 문서가 UEM 기준본이며, 외부 자료는 참조용이다.**

## 1. 기본 구조
- 음절 σ = (C, V, F, I)
  - C (초성): 구조/객체 조작자.
  - V (중성): 방향/시간/사영 연산자.
  - F (종성): 경계/여백/복소화 주석 연산자.
  - I (Index): 위치/자릿수 메타(옵션, 필요 시 사용).
- 해석 함수 Γ: (C,V,F) ↦ 연산자 O ∈ Ops.  
  합성 규칙: J(C,V,F)(X) = A_F ( V ( C ( X ) ) )  
  - C: X ↦ 구조 변환/선택/조합.
  - V: 방향/시간/사영·미분 연산.
  - F: 경계/여백 주석·종결·복소화.
- 전역 총함수 요구: 모든 합법적 (C,V,F) 조합은 해석 가능해야 하며 타입이 정해진다(에러는 미리 정의된 금지 조합에서만 발생).

## 2. 연산자 의미(초/중/종성)
- C (예시 역할 스켈레톤)
  - ㄱ: 구조 생성/선택, ㄴ: 포함/내장, ㄹ: 경계 확장, ㅂ: 부피/객체 변환, ㅅ: 분할/선택.
- V (방향/시간/사영 스켈레톤)
  - ㅏ: +x 방향 미분/사영 D_{x+}
  - ㅓ: -x 방향 미분/사영 D_{x-}
  - ㅑ: +t 시간 미분 ∂_t
  - ㅕ: -t 시간 미분 -∂_t
  - ㅐ: 시공간 결합 D_{x+} + λ ∂_t (λ는 사전 정의 또는 컨텍스트)
  - ㅣ: 시간 적분 ∫ dt (지속)
  - 합성 모음(ㅚ 등)은 선형/조건부 결합으로 해석한다(표로 고정 예정).
- F (경계/여백/주석 스켈레톤)
  - ㅁ: 여백 생성/보존 주석
  - ㅿ/ㅎ: 경계·불확정/질의 주석
  - 겹받침: 경계 결합/복소화 또는 차원 승격(예: ㄹㅎ → 경계+기울기)
- **TODO 표기**: 위 역할은 스켈레톤이다. UEM 본문 표를 별도 작성해 (C,V,F)→연산 서명/도메인/코덱을 확정해야 한다.

## 3. 초병렬 연산
- 병렬 합성 ⊗_par : Ops × Ops → Ops_par
  - 결합성 필요, 단위원 e_par 정의(없으면 명시적으로 none).
  - 충돌 해결 규칙 resolve(annot_left, annot_right) → annot ∪ Error 는 결정적이어야 함.
  - 타입 보존: 입력이 well-typed면 출력도 동일 도메인/타입을 유지.
- 다방향 발산 ∇_hangul f := ⊗_par { ∂_γ f | γ ∈ Γ-basis }
  - Γ-basis는 선택된 방향 기저(예: {ㅏ, ㅑ, ㅕ, ㅐ, ㅣ, …})로 명시.
  - HS 축소판: (∂_x, ∂_t, ∂_domain) 같은 제한 기저를 표로 고정.
- 로그/여백 기록: ⊗_par와 ∇_hangul 적용 시 제거·무시된 성분은 MarginLog에 기록(“source, context, value, weight, entropy”).

## 4. 표/정책(작성 지침)
- C/V/F LUT: JIWOL 표(초19/중21/종28)를 베이스로 UEM 의미열을 붙여 작성.
- 금지 조합: 종성 금지/겹받침 정책, 초성 겹자음 금지 등 STRICT/TOTAL 정책을 표로 명시.
- 오류 코드: E_L_EMPTY, E_V_EMPTY, E_T_BAD_COMBO 등 명확히 열거.
- HS 서브셋: 교육용 최소 기저/규칙을 별도 표로 제공.

## 5. Lean/코드 매핑(후속)
- 선언만: `structure HangulOp := (C : …) (V : …) (F : …)` 형태와 `Gamma : C → V → F → Op`.
- 증명 필요 성질: ⊗_par 결합성/타입 보존, ∇_hangul 기저 독립성, J(C,V,F) 타입 보존/정규형 존재 여부.
- 매핑 테이블: 스펙 기호(Π, τ_margin 등) ↔ Lean 이름/notation을 별도 표로 추가.
